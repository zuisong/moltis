開始使用

安裝 Moltis，幾分鐘內啟用 AI 助理

~ 快速開始

$ curl -fsSL https://www.moltis.org/install.sh | sh

$ brew install moltis-org/tap/moltis

# 多架構：amd64/arm64

$ docker run -d \

--name moltis -p 13131:13131 -p 1455:1455 \

-v moltis-config:/home/moltis/.config/moltis \

-v moltis-data:/home/moltis/.moltis \

-v /var/run/docker.sock:/var/run/docker.sock \

ghcr.io/moltis-org/moltis:latest

# 開啟 http://localhost:13131 並完成設定

# 從原始碼建置

$ cargo install moltis --git https://github.com/moltis-org/moltis

# 從 GitHub Releases

# Debian/Ubuntu (.deb)

$ sudo dpkg -i moltis_*_amd64.deb

# Fedora/RHEL (.rpm)

$ sudo dnf install moltis-*.x86_64.rpm

# Arch Linux (.pkg.tar.zst)

$ sudo pacman -U moltis-*-x86_64.pkg.tar.zst

# Snap

$ sudo snap install moltis

# AppImage

$ ./moltis-*-x86_64.AppImage

身分驗證

SSRF 防護

封鎖 LLM 擷取時存取 loopback、私有與 link-local IP。

提供外掛、Hook，以及支援 OAuth 2.1 的 MCP 工具伺服器。支援 stdio 或 HTTP/SSE，並可自動重新啟動。

受 Pi 啟發的自我延展

安全優先設計

您可以信賴的機器 AI 助手

為什麼您應該信任它？

Moltis 是一個檔案。您下載它，執行它，完成。沒有外掛商店，沒有套件管理員，沒有可能被破壞的執行環境。整個應用程式是一個您可以驗證的單一二進位檔案。

您的 AI 預設在沙盒中執行。除非您明確允許，否則它無法讀取您的檔案、執行指令或瀏覽網頁。每個工具操作都可以要求您的核准 — 或者您可以預先核准您信任的特定操作。

您的密碼和 API 金鑰會以保護容器包裹，因此永遠不會被意外記錄，並且在不再需要時記憶體會被歸零。它所使用的程式語言（Rust）在編譯器層級消除了整類安全漏洞。

這裡您找不到的

沒有可能被攻破的外掛市場。沒有 JavaScript 執行環境。沒有 1,200 個您無法審查的套件的相依樹。一個您可以從頭到尾稽核的二進位檔案。
